Getting Connected

New Launcher

If you have the new launcher, go to settings -> Login Servers -> Add

Name: localserver

Host: {{ server_status.server_ip }}

Port: {{ server_status.login_port }}

Hit [OK] and [OK]

Choose "localserver" in the dropdown next to the [START] button.


NOTE: Autoregistration is enabled, to create a new account enter any username you desire and the new password. That account will be created on the fly as you login and this user/password combination will allow you to return to your characters.

Hit the [START] button and login.
